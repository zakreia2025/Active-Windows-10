# Active-Windows-10

A simple tool to activate Windows 10/11 permanently using KMS.

---

## 📥 Download

Click the green **"Code"** button → **"Download ZIP"**  
Or clone the repository:

```bash
git clone https://github.com/zakreia2025/Active-Windows-10.git

🚀 How to Use (Windows)

1. Extract the files

Unzip the downloaded folder to your desktop or any folder.

2. Run as Administrator

Right-click on Activating-Windows-10.cmd and select "Run as Administrator".

3. Enter password

When prompted, type:

```
abuamer
```

(The password will not appear on screen while typing)

4. Wait

The tool will:

· Check Windows activation status
· Activate Windows automatically (if needed)

5. Done!

Windows is now activated permanently.

---

🔍 How to check if Windows is activated?

Open CMD as Administrator and type:

```cmd
slmgr /xpr
```

If you see "The machine is permanently activated" → Done ✅
